## 7.1 結構化程式設計 ##

* 三種結構化程式設計
  1. 循序性結構(sequence structure): 採上至下(top to down)
  2. 選擇性結構(selection structure): if-else, switch
  3. 重複性結構(iteration structure): for, while, do-while

* 單一的出入口，使程式易讀、好維護，以減少除錯的時間。
