## 11.1 認識結構 ##

* 陣列只能存放相同資料型態的資料，結構可存放不同資料型態的資料。

* 利用「結構成員存取運算子」或稱「點號運算子.」存取變數內的成員。

* "結構變數所佔的位元組"會以"結構成員裡，所佔位元組最多的資料型態的倍數"來配置記憶體空間。
