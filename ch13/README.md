## 13.1 程式與模組化與實作 ##

* 程式的模組化
  * 發展大型程式中，可以將函數分開來撰寫，依其功能來區分，將一個或數個函數模組存成一個檔案，然後分別編譯，等到所有的函數模組都開發完畢後，再與主程式一起連接與執行。

## 13.2 於不同檔案裡使用全域變數 ##

* 將程式模組化，獨立成數個不同的檔案後，如果想在不同的檔案裡使用同一個全域變數，則必須在函數裡把該變數利用 C 語言的關鍵字 extern 來宣告。

* 宣告於主程式中的全域變數，與宣告於 .c 檔中 extern 的同名變數，是同一個變數，因此會共用同一個記憶體空間。

## 13.3 條件式編譯 ##

* 希望編譯器在某些情況下，只將程式中的某個部分編譯，而不是全部都編譯。

* 前置處理指令
  * #ifdef
  * #else
  * #endif
  * #ifndef
  * #if
  * #else
  * #elif
  * #endif
  * #undef

## 13.4 命令列引數的使用 ##

* 格式
  * int main(int argc, char *argv[]){}
  * argc
    * argument count
  * argv
    * argument value
    * 指標陣列
    * 系統自動將 argv[0] 指向執行檔的檔名(包含路徑)
    * 因為無法事先得知使用者會由命令列中輸入多少個引數，在宣告時，不需要限定陣列的大小。
