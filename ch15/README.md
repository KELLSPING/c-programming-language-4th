## 15.4 位元運算子 ##

### 15.4.1 位元運算子 ###

* C 語言提供了"位元邏輯"與"位元位移"兩種運算子。
* 位元邏輯運算子和邏輯運算子是不相同的。
  * 位元邏輯運算子: 是作用到整數裡的每一個位元。
    * ~ (NOT)
    * & (AND)
    * | (OR)
    * ^ (XOR)
  * 邏輯運算子: 把整個數值看成一個單位，然後再作用到數值上。
    * && (AND)
    * || (OR)

### 15.4.2 位元移位運算子 ###

* 位元移位運算子只有在整數型態才能運算。
* 左移運算子 (<<)
  * 超果最左邊的位元予以刪除，而空白位數則補 0。
* 右移運算子 (>>)
  * 超果最右邊的位元予以刪除，而空白位數則補 0。

## 15.5 位元欄位 ##

* C 語言提供一種特別的結構，能夠充分使用到結構變數中的每一位元，這種特別的結構稱為位元欄位結構。
* 資料型態必須是整數 (int) 或無號整數 (unsigned)，通常位元欄位是以無號整數來宣告位元欄位的型態。

```C
struct 位元欄位結構的名稱{
    資料型態 欄位名稱1: 位元長度;
    資料型態 欄位名稱2: 位元長度;
    ...
    資料型態 欄位名稱n: 位元長度;
};
```

* 例如，定義一個位元欄位結構，其欄位包括性別 (sex，1 個位元)、婚姻狀況 (marriage，1 個位元) 以及年齡 (age，7 個位元)

```C
struct statue{
    unsigned sex:1;
    unsigned marriage:1;
    unsigned age:7;
};
```

* 位元欄位結構的變數佔有 4 個位元組。
* 使用位元欄位的結構可以節省記憶體空間。
